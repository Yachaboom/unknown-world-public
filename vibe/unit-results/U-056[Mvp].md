## 1. 작업 요약

인벤토리 패널에서 아이템 이름이 길 경우 잘리는 문제를 시각적으로 처리(ellipsis)하고, 마우스 호버 시 브라우저 네이티브 툴팁(`title`)을 통해 전체 이름과 수량을 명확히 확인할 수 있도록 개선하였습니다.

---

## 2. 작업 범위

- **텍스트 오버플로우 처리**: CSS `text-overflow: ellipsis`와 `display: block`을 적용하여 긴 텍스트가 영역을 벗어나지 않도록 시각적 처리
- **네이티브 툴팁 구현**: `InventoryPanel`의 각 아이템 요소에 `title` 속성 추가 (수량 1 초과 시 "이름 x 수량" 포맷 적용)
- **단위 테스트 추가**: 툴팁 표시 여부 및 수량 포함 포맷 검증 테스트 작성
- **문서 업데이트**: 작업 방식 변경(커스텀 툴팁 -> 네이티브 툴팁)에 따른 유닛 계획서 갱신 및 완료 처리

- `title` 속성을 동적으로 생성하여 주입: `item.quantity > 1 ? `${item.name} x ${item.quantity}` : item.name`
