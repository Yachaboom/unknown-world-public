# U-058[Mvp] 핫스팟 디자인 개선

## 개요

- **목표**: 핫스팟(Hotspot) 시각적 디자인을 게임 UI 수준으로 개선
- **의존성**: U-010[Mvp] (Scene Canvas + Hotspot Overlay)
- **우선순위**: MVP 품질 개선

## 배경

현재 핫스팟 디자인이 단순하여 게임 UI로서의 품질이 부족함. 터미널 테마에 맞으면서도 눈에 띄는 디자인으로 개선 필요.

## 작업 내용

### A. 핫스팟 시각적 개선

- **코너 디자인**: 핫스팟 영역의 4개 코너에 스타일리시한 마커/브라켓 추가
- **얇은 스트로크**: 전체 테두리 대신 미세한 외곽선으로 영역 표시
- **옅은 배경**: 반투명 배경으로 클릭 가능 영역 암시

### B. 색상 설계

- 터미널 컬러 팔레트 내에서 녹색이 아닌 대체 강조 색상 선택
  - 후보: cyan/teal 계열, amber/gold 계열, 또는 magenta 계열
- 호버/포커스 상태에 따른 색상 변화
- 비활성화 상태 시각화

### C. 인터랙션 피드백

- 호버 시 코너 마커 애니메이션 (미세한 펄스 또는 확장)
- 클릭/드롭 타겟 시 하이라이트 강화
- 비활성 핫스팟의 시각적 구분

## 완료 기준 (DoD)

- [ ] 핫스팟이 코너 마커 + 얇은 스트로크 + 옅은 배경 스타일로 표시
- [ ] 녹색이 아닌 터미널 호환 색상으로 강조됨
- [ ] 호버/포커스 상태에서 시각적 피드백 제공
- [ ] DnD 드롭 타겟으로서의 시각적 명확성 확보
- [ ] 기존 bbox 좌표 규약(0~1000) 준수
