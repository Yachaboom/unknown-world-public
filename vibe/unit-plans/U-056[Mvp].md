# U-056[Mvp] 인벤토리 아이템 이름 텍스트 잘림 최소화

## 개요

- **목표**: 인벤토리 아이템 이름이 잘리는 문제를 최소화하고, 잘린 경우 툴팁으로 전체 이름 표시
- **의존성**: U-011[Mvp] (Inventory 패널 DnD 기본)
- **우선순위**: MVP 품질 개선

## 배경

현재 인벤토리 패널에서 아이템 이름이 길 경우 잘려서 표시되어 사용자가 전체 이름을 확인할 수 없음. 게임 UX 관점에서 아이템 식별이 중요하므로 개선 필요.

## 작업 내용

### A. 폰트 크기 조정

- 인벤토리 아이템 이름 폰트 크기 미세 조정 (가독성 유지하면서 공간 효율화)
- CSS 변수로 관리하여 일관성 유지

### B. 텍스트 오버플로우 처리

- `text-overflow: ellipsis` 적용으로 잘린 텍스트 시각적 표시
- 텍스트가 잘렸는지 동적으로 감지하는 유틸리티 훅 구현

### C. 툴팁 구현

- 잘린 이름인 경우에만 마우스 오버 시 툴팁 표시
- 기존 CRT 테마와 일관된 스타일링
- 접근성(aria-label) 고려

## 완료 기준 (DoD)

- [ ] 긴 아이템 이름이 ellipsis로 잘려서 표시됨
- [ ] 잘린 아이템 호버 시 전체 이름이 툴팁으로 표시됨
- [ ] 잘리지 않은 아이템은 툴팁이 표시되지 않음
- [ ] CRT 테마와 일관된 툴팁 스타일
