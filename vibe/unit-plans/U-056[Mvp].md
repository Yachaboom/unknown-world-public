# U-056[Mvp] 인벤토리 아이템 이름 텍스트 잘림 최소화

## 개요

- **목표**: 인벤토리 아이템 이름이 잘리는 문제를 최소화하고, 툴팁으로 전체 이름 표시
- **의존성**: U-011[Mvp] (Inventory 패널 DnD 기본)
- **우선순위**: MVP 품질 개선

## 배경

현재 인벤토리 패널에서 아이템 이름이 길 경우 잘려서 표시되어 사용자가 전체 이름을 확인할 수 없음. 게임 UX 관점에서 아이템 식별이 중요하므로 개선 필요.

## 작업 내용

### A. 텍스트 오버플로우 처리

- `text-overflow: ellipsis` 적용으로 잘린 텍스트 시각적 표시
- CSS 변수로 관리하여 일관성 유지

### B. 네이티브 툴팁 구현

- 모든 아이템에 마우스 오버 시 브라우저 네이티브 툴팁(`title` 속성) 표시
- 수량이 1보다 큰 경우 "아이템명 x 수량" 형식으로 표시
- 기존 접근성(aria-label) 유지

## 완료 기준 (DoD)

- [x] 긴 아이템 이름이 ellipsis로 잘려서 표시됨
- [x] 아이템 호버 시 전체 이름이 네이티브 툴팁으로 표시됨
- [x] 수량이 복수인 경우 툴팁에 수량 포함
